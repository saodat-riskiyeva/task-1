HTML & CSS Basics Summary

HTML Summary

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It uses tags to structure content, such as headings, paragraphs, links, and images. HTML documents are made up of elements that define the structure and layout of a webpage.

HTML Elements

HTML elements are the building blocks of web pages. They consist of a start tag, content, and an end tag. Common elements include:

  • <h1> to <h6> for headings
  • <p> for paragraphs
  • <a> for links
  • <img> for images
  • <div> for divisions or sections

CSS Summary

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. It allows you to control the layout, colors, fonts, and overall appearance of web pages. CSS can be applied inline, embedded in the HTML document, or linked as an external stylesheet.
